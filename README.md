# geometry-volume-app-course
Simulate a simple geometry volume calculator

# Descripción del proyecto
El proyecto contiene dos carpetas: 
 - geometry: Códigos para calcular el volumen de cubos, esferas, cilindros y conos. 
 - test: Códigos que se aseguran de que los códigos de la carpeta geometry funcionen correctamente. 

# Estructura del proyecto
geometry-volume-app-course/
│-- geometry/
│ │-- box.py
│ │-- cone.py
│ │-- cylinder.py
│ │-- sphere.py
│ │-- init.py
│-- tests/
│ │-- test_box.py
│ │-- test_cone.py
│ │-- test_cylinder.py
│ │-- test_sphere.py
│ │-- init.py
│-- main.py
│-- README.md
│-- requirements.txt

# Como correr el proyecto
Para correr el proyecto usa py main.py en la terminal de bash

Para correr los tests usa py -m pytest

# Dependencias
Se requiere pytest
